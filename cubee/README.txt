CuBee Drop 規約ページセット（SakuLink）

このフォルダには、CuBee Drop の「規約等」ページ一式が入っています。
- policy.html   … 規約等の入口（利用規約・プライバシー・権利表記へのリンク）
- terms.html    … 利用規約
- privacy.html  … プライバシーポリシー
- legal.html    … 権利表記 / クレジット
- index_patch.txt … メニュー画面へ「規約等」を追加する1行（2パターン）

【設置場所】
GitHub Pages の /cubee/ 配下に置く想定です（例：リポジトリの cubee フォルダ、または root 直下が /cubee/ の場合は同階層）。

【動作確認URL】
https://sakulink.com/cubee/policy.html

【補足】
このひな形は“最小限の短い文章”です。アクセス解析やランキング等でデータ取得を追加した場合は、privacy.html に追記が必要です。
